2.0-Testing9 26/01/12 (Versión Rejected)

  • Problems with Calendar, where an error message appears, and makes impossible to select a date.
  • The bug reported “4314” is not resolved. This bug does not show the size of the files to upload, besides not knowing where the CV´s are stored. The development department was suggested to take out this options from User Tab.
  • The groups that were created in an English environment, are not visualized in the spanish environment.
  • The option Reassign causes malfunction problems.
  • The dates greater than 1970, can not be select in the option Static Date, inside Date.

 

  • When you save a form without a button, you should change the message, suggested in the bug.

 

  • The event option does not display the labels, in addition to not have made the change to the text suggested in the bug.

 

  • Problems with triple units with grids.

 

  • Problems with summations in grids.

  • The event option does not display the labels, in addition to not have made the change to the text suggested in the bug.
